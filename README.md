# Django Music Library
> This is a simple music library implemented using Django class based views.

## Built With
- Bootstrap
- Python 
- Django

## Getting Started
- Run `git clone git@github.com:Tamunokuro/DJANGO-QUIZAPP.git`

### Prerequisites
- Python 3.10
- Django 4.1
